# E203: E203: Invalid form type marker

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Word contains an invalid or undeclared form type marker (e.g., has multiple stacked markers).

## Examples

### Example 1

@UTF8
@Begin
@Languages:	eng
