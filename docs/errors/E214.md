# E214: E214: Empty scoped annotation content

**Severity**: error

## Description

A scoped annotation (e.g., error annotation , replacement ) hasan empty content list. The validator reports E214 when annotated content haszero scoped annotations attached.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Comment:	ERROR: Scoped annotation cannot be empty
@Comment:	Invalid: 'hello [*]' - Empty error annotation
*CHI:	hello [*] .
@End
```
