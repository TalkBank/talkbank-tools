# E233: E233: Empty compound part

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Compound markers () must connect two non-empty parts. Adjacent compound markers() create an empty part between them, which is invalid.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Comment:	ERROR: Adjacent compound markers create empty compound part
@Comment:	Invalid: 'un++do' - Empty part between compound markers
*CHI:	un++do .
@End
```
