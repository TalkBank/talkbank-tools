# E241: E241: Illegal Untranscribed Marker 'xx'

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The marker 'xx' is used for untranscribed speech, but this is not allowed in CHAT. The correct marker for untranscribed speech is 'xxx' (three x's).

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	I said xx today .
@End
```
