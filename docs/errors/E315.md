# E315: E315: Invalid control character

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Main tier or dependent tier contains an invalid control character (e.g., embedded NUL, SOH, or other non-printable ASCII).
