# E341: E341: UnbalancedQuotationCrossUtterance

> ⏳ Planned — This check is documented but not yet enforced by the validator. The error code will not fire until implementation is complete.

**Severity**: error

**Status**: ⏳ Planned

## Description

A quotation-follows terminator () is used but the next utterancefrom the same speaker does not begin with a quotation precedes linker(). This indicates an unbalanced cross-utterance quotation sequence.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	MOT Mother, CHI Target_Child
@ID:	eng|test|MOT||female|||Mother|||
@ID:	eng|test|CHI||male|||Target_Child|||
*MOT:	she said +"/.
*CHI:	what .
*MOT:	go to bed .
@End
```
