# E357: E357: UnmatchedUnderlineEnd

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

An underline end marker was found without a preceding underline beginmarker in the same utterance. The end marker has no open underline toclose.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Comment:	ERROR: Unmatched underline end marker
*CHI:	hello world .
@End
```
