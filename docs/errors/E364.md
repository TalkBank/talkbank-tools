# E364: E364: Malformed word content

> ⏳ Planned — This check is documented but not yet enforced by the validator. The error code will not fire until implementation is complete.

**Severity**: error

**Status**: ⏳ Planned

## Description

Word content is structurally malformed — the parser recognized a word node but its internal structure is invalid (e.g., with instead of a language code).

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Child
@ID:	eng|corpus|CHI|||||Child|||
```
