# E365: E365: Malformed tier content

> ⏳ Planned — This check is documented but not yet enforced by the validator. The error code will not fire until implementation is complete.

**Severity**: error

**Status**: ⏳ Planned

## Description

A header or tier has content that does not match any recognized CHAT headerstructure. The parser reports E365 when it encounters an unknown node typeduring header dispatch in the CST.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Child
@ID:	eng|corpus|CHI|||||Child|||
*CHI:	hello .
%pho:	***bad***
@End
```
