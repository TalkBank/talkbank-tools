# E370: E370 — Structural order error

> ⏳ Planned — This check is documented but not yet enforced by the validator. The error code will not fire until implementation is complete.

**Severity**: error

**Status**: ⏳ Planned

## Description

A structural ordering violation in the utterance content, such as groups orreplacements that do not align correctly with dependent tier items.
