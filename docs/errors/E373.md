# E373: E373: InvalidOverlapIndex

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

An overlap marker has an index value outside the valid range. For CAoverlap brackets (), the index must be 2–9. For scoped overlapannotations (, ), the index must be 1–9.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Comment:	ERROR: Overlap index must be 2-9
*CHI:	⌈1 hello ⌉ .
@End
```
