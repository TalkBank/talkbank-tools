# E373: E373: InvalidOverlapIndex

**Severity**: error

## Description

An overlap marker has an index value outside the valid range. For CAoverlap brackets (), the index must be 2–9. For scoped overlapannotations (, ), the index must be 1–9.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	MOT Mother, CHI Target_Child
@ID:	eng|test|MOT||female|||Mother|||
@ID:	eng|test|CHI||male|||Target_Child|||
*MOT:	I think [<0] so .
*CHI:	yeah [>0] .
@End
```
