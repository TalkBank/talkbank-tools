# E502: E502: Missing required @End header

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

Every valid CHAT file must end with an header. This error indicates the file is missing , usually because the file is truncated, empty, or was saved incompletely.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Child
@ID:	eng|corpus|CHI|||||Child|||
*CHI:	hello world .
```
