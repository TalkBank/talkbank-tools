# E534: E534: Unsupported @Options Value

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

An header contains a flag that is not one of the recognized option values. The file parses successfully but the unsupported flag is stored as and flagged during validation.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Options:	badoption
*CHI:	hello world .
@End
```
