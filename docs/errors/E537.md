# E537: E537: Unsupported @Number Value

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

An header contains a value that is not one of the recognized number options. The file parses successfully but the unsupported value is stored as and flagged during validation.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Number:	badnumber
*CHI:	hello world .
@End
```
