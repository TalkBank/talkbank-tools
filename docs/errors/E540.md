# E540: E540: Invalid @Time Duration Format

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

An header contains a value that does not match the expected time format. The file parses successfully but the invalid value is stored as and flagged during validation.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Time Duration:	not-a-time
*CHI:	hello world .
@End
```
