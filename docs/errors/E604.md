# E604: E604: %gra Tier Without %mor Tier

**Severity**: error

## Description

A %gra (grammatical relations) tier appears without a corresponding %mor (morphology) tier. According to CHAT rules, %gra depends on %mor and cannot exist independently.

## Examples

### Example 1
