# E704: overlap — overlapping overlap markers

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

A single speaker has consecutive utterances with overlap markers (/) that overlap with each other. Overlap markers should indicate simultaneous speech between different speakers, not self-overlap.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
*CHI:	hello ⌈ world ⌉ .
*CHI:	⌊ testing ⌋ .
@End
```
