# E707: E707: Mor terminator presence mismatch

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The %mor tier has a terminator but the main tier does not, or vice versa.One tier ends with a sentence-final punctuation mark while the other does not.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|test|CHI||female|||Target_Child|||
*CHI:	hello .
%mor:	co|hello
@Comment:	ERROR: Main tier has terminator "." but %mor has none
@End
```
