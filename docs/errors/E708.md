# E708: E708: Malformed grammar relation on %gra tier

> ⏳ Planned — This check is documented but not yet enforced by the validator. The error code will not fire until implementation is complete.

**Severity**: error

**Status**: ⏳ Planned

## Description

A grammar relation on the tier is malformed — missing an index, head,or relation label, or containing non-integer values where integers are expected.The tier format is for each word.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Child
@ID:	eng|corpus|CHI|||||Child|||
*CHI:	hello world .
%gra:	|2|SUBJ 2|0|ROOT
@End
```
