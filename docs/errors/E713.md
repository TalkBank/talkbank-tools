# E713: E713: Gra head index invalid

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

A relation has a head index that falls outside the valid range, where is the number of chunks in the utterance. Index is reserved for the ROOT head; otherwise the head index must point toan existing chunk.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|||||Target_Child|||
@Comment:	ERROR: head index 9 is out of range for 3 mor chunks
*CHI:	I want .
%mor:	pro|I v|want .
%gra:	1|2|SUBJ 2|0|ROOT 3|9|PUNCT
@End
```
