# E714: E714:  alignment count mismatch — too few tokens

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The (actual phonology) tier has fewer alignable tokens than the main tier.Each main-tier word must have a corresponding token.

## Examples

### Example 1

```chat
@UTF8
@Begin

*CHI:	I want cookie .
%pho:	aɪ wɑnt
```
