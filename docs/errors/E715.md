# E715: too many tier tokens

**Severity**: error

## Description

Alignment count mismatch: a pho/mod/wor tier has more alignable items than the main tier.

## Examples

### Example 1

@Begin
@Languages:	eng

*CHI:	want cookie .
%pho:	aɪ wɑnt kʊki
@Comment:	ERROR: Main tier has 2 words but %pho has 3 tokens (extra aɪ)
