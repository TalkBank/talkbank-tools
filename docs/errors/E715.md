# E715: E715:  alignment count mismatch — too many tokens

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The (actual phonology) tier has more alignable tokens than the main tier.Remove the extra tokens so counts match.
