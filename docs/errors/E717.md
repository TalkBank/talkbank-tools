# E717: E717: Pho terminator presence mismatch

**Severity**: error

## Description

The %pho tier has a terminator but the main tier does not, or vice versa.One tier ends with a sentence-final punctuation mark while the other does not.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|test|CHI||female|||Target_Child|||
*CHI:	hello .
%pho:	hɛloʊ
@Comment:	ERROR: Main tier has terminator "." but %pho has none
@End
```
