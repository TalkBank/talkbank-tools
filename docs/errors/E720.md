# E720: Gra count mismatch

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The number of chunks does not equal the number of relationsfor an utterance. aligns 1-to-1 with chunks (not items — a item with post-clitics produces multiple chunks).

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng
@Participants:	CHI Target_Child
@ID:	eng|corpus|CHI|2;6|male|||Target_Child|||
@Comment:	Note: %gra aligns to %mor chunks, not items!
*CHI:	I want cookie .
%mor:	pro|I v|want n|cookie .
%gra:	1|2|SUBJ 2|0|ROOT 3|2|OBJ 4|4|EXTRA 5|2|PUNCT
@Comment:	ERROR: %mor has 4 chunks (3 words + terminator) but %gra has 5 relations
@End
```
