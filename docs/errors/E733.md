# E733: E733:  alignment count mismatch — too few tokens

> ✅ Active — This check is active in the validator.

**Severity**: error

**Status**: ✅ Active

## Description

The (model/target phonology) tier has fewer alignable tokens than themain tier. Each main-tier word must have a corresponding token.

## Examples

### Example 1

```chat
@UTF8
@Begin
@Languages:	eng

*CHI:	I want cookie .
%mod:	aɪ wɑnt
@Comment:	ERROR: Main tier has 3 words but %mod only has 2 tokens (missing cookie)
@End
```
